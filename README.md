# RE Omnitrix
Ben 10 (classic) aliens classification windows app.

## Download Source Code
```bash
git clone https://github.com/playatanu/watercolor-app re-omnitrix

cd re_omnitrix
```
## Install Packages
```bash
pip install -r requirements.txt
```
## Run
```python
python main.py
```
